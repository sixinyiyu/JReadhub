* 类加载机制

  1. 加载（获取来自任意来源的字节流并转换成运行时数据结构，生成Class对象）
  2. 验证（验证字节流信息符合当前虚拟机的要求，防止被篡改过的字节码危害JVM安全）
  3. 准备（为类变量分配内存并设置初始值）
  4. 解析（将常量池的符号引用替换为直接引用，符号引用是用一组符号来描述所引用的目标，直接引用是指向目标的指针）
  5. 初始化（执行类构造器、类变量赋值、静态语句块）
